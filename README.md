## Objective

A portfolio-driven game development hub demonstrating end-to-end Unity engineering — from single-player demos and multiplayer architecture to IAP monetization and VR co-op experiences. Built following agile practices with structured sprints, living documentation, and versioned releases.

---

| Phase | Sprints | Focus | Milestone |

| **F1 · Demos** | S1–S6 | 8 demos across 2D, 3D, VR, AR, and multiplayer | Publish on itch.io |
